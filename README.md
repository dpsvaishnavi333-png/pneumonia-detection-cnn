# Pneumonia Detection Using CNN

This project uses a Convolutional Neural Network (CNN) trained with TensorFlow/Keras to detect pneumonia from chest X-ray images.

## Features
- CNN with Dropout layers
- Data augmentation
- 40 epochs training
- Accuracy & loss visualization
- Final .keras model saved

## Technologies Used
- Python
- TensorFlow
- Matplotlib
- Deep Learning

## How the Model Works
The model uses multiple convolution + max pooling layers and predicts binary output (Pneumonia / Normal).

Full code is available in `pneumonia_cnn.py`.
